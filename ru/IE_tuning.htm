Настройка MS Internet Explorer

При просмотре через веб-страницу как через двусторонний канал, так и через односторонний, в настройках безопасности MS Internet Explorer должны быть разрешены:

   — работа сценариев JavaScript,
   — запуск ActiveX-компонентов сторонних производителей.

Для разрешения работы сценариев JavaScript и запуска ActiveX компонентов сторонних производителей следует войти в меню MS Internet Explorer:

"Tools"→"Internet Options"→"Security"→"Internet"

(В русскоязычной версии MS Internet Explorer:

"Сервис"→"Свойства обозревателя"→"Безопасность"→"Зона Интернета")

Следует нажать кнопку "Default Level" ("По умолчанию") и затем с помощью бегунка выставить уровень безопасности в "Medium" ("Средний").

Затем, нажав "Custom Level" ("Другой"), установить следующие настройки:

Download unsigned ActiveX controls
(Загрузка неподписанных ActiveX-компонентов

Prompt
 Предлагать)

Automatic prompting ActiveX controls
(Автоматически спрашивать для неподписанных компонентов ActiveX

Enable
Разрешить)

Выйдите из панели, нажав кнопку "OK".

Подтвердите настройки, нажав кнопку "Yes" ("Да").

Текущий пользователь при установке ActiveX-компонента (при первой загрузке) должен иметь права администратора.

Вход на страницу просмотра

Через несколько секунд после входа MS Internet Explorer на веб-страницу с видеоокном появится вопрос

Следует нажать "Install" ("Установить").

Требования к компьютерной сети

Для приема видеопотока необходимо, чтобы на промежуточных узлах сети между сервером, передающим видео-аудиопоток и на клиентской машине было разрешено прохождение потока данных по порту 80 по протоколам TCP и UDP в обе стороны.
